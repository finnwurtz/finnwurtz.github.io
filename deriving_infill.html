Deriving an Infill Growth Index

Overview

This page explains and plots the metrics used to calculate the infill growth index proposed in my Perceptual Density Project, using data from the city of Seattle, Washington. The infill growth index (Figure 4) is a metric of urban infill growth potential, intended as a proxy for block groups’ sensitivity to infill growth, that either directly or indirectly incorporates the following measures:

• Gross density (Figure 1) is the number of households in a block group divided by the block group’s land area.

• Composite density (Figure 2) is a hybrid metric of urban density, intended as a proxy for perceived neighborhood density.

• Perceptual offset (Figure 3) is the degree to which a block group’s composite density percentile either overperforms or underperforms its gross density percentile, intended as a proxy for the disconnect between a block group’s gross density and the perceived density of the neighborhood.

Figure 1: Seattle Gross Density

To derive an infill growth index, we again start with gross density, or the density of households per land area for each of the block groups in the study area. This plot displays the gross household density distribution for each of the block groups analyzed in the Seattle test case, with the red zones corresponding to the block groups with the highest densities and the blue zones corresponding to the block groups with the lowest densities. Note that each color contains approximately the same proportion of the study area’s households; hence, the total area of the red zones is much smaller than the total area of the blue zones.

Figure 2: Seattle Composite Density

Next, we look at gross density alongside composite density, which is itself a combination of gross density, travelshed density, and demic unit radius values. I explain how I derive composite density on this page. This plot displays the composite density distribution for the Seattle test case, with the red zones again corresponding to the block groups with the highest densities and the blue zones again corresponding to the block groups with the lowest densities.

Figure 3: Seattle Perceptual Offset

We get our third variable, perceptual offset, by calculating the household-weighted distribution of block groups by gross density and the household-weighted distribution of block groups by composite density. The perceptual offset of a block group is the difference between a block group’s composite density percentile and its gross density percentile. Hence, block groups with lower gross household density values relative to their neighbors tend to have higher perceptual offset values. This plot displays the perceptual offset distribution for the Seattle test case, with the red zones corresponding to the block groups with the highest values and the blue zones corresponding to the block groups with the lowest values.

Figure 4: Seattle Infill Growth Index

Finally, to get the infill growth index value of each block group, we average its household-weighted composite percentile value with its household-weighted perceptual offset percentile and take the percentile value of the result. It follows that the block groups with the highest infill growth index values tend to have both high composite density values and high perceptual offset values, meaning that they are surrounded by high-density block groups and are not as dense as their neighbors. This plot displays the infill growth index distribution for the Seattle test case; as before, the red zones again corresponding to the block groups with the highest values and the blue zones again corresponding to the block groups with the lowest values.
